El patrimonio cultural mundial a un solo click.

¡Pruébalo!

Navega por el mapa. Descubre joyas culturales.

Con un simple navegador tienes acceso en la punta de tus dedos a datos masivos de patrimonio cultural. Navega por el mapa, vete a sitios remotos o cercanos y encuentra lugares únicos. LOD4Culture está optimizado a conciencia para que funcione incluso en dispositivos móviles. Navegadores recomendados: Firefox y Chrome.

Más o menos zoom. Pruébalo tú mismo.

Explora el mapa a diferentes niveles de zoom para visualizar un área amplia o para descubrir los tesoros culturales de un barrio. Filtra los sitios por tipo (por ejemplo Museos de arte, Castillos, Dólmenes...) para encontrar las cosas que te interesan.

Información cultural detallada. Sitios, obras de arte y artistas.

Más allá del mapa, navega por la enorme colección de recursos de patrimonio cultural disponibles. LOD4Culture te permite encontrar datos relevantes de sitios, obras de arte y artistas. Hay muchísima información para analizar: imágenes, descripciones, localizaciones, fechas, arquitectos, listas de colecciones... y muy fácilmente accesibles.

Basado en Datos Abiertos Enlazados. Jaque mate.

LOD4Culture extrae Datos Abiertos Enlazados de nuestros proveedores de datos preferidos: Wikidata y DBpedia. No es nada fácil utilizar Datos Abiertos Enlazados; hemos domado a la bestia utilizando CRAFTS.

Wikidata

Wikidata es un grafo de conocimiento multilingüe que se edita de manera colaborativa. Es una fuente de datos abiertos típica de proyectos de Wikimedia como Wikipedia, y que cualquiera puede utilizar al basarse en la licencia CC0 de dominio público. Wikidata contiene datos de patrimonio cultural de altísima calidad y es la principal fuente usada en LOD4Culture.

DBpedia

DBpedia es un proyecto dedicado a extraer datos estructurados a partir de la Wikipedia. Esta información estructurada se publica para que cualquiera pueda consultarla. DBpedia es una fuente de información secundaria de LOD4Culture, empleada para extraer descripciones textuales y categorías de Wikipedia acerca de recursos de patrimonio cultural.

CRAFTS

Acceder a Datos Abiertos Enlazados no es una tarea para nada sencilla. LOD4Culture utiliza CRAFTS para simplificar el acceso a los datos culturales de Wikidata y DBpedia. Hemos configurado una API de CRAFTS que se encarga de traducir de manera automática las llamadas a la API en consultas a Wikidata y DBpedia. CRAFTS y LOD4Culture son creaciones originales de Guillermo Vega Gorgojo.
